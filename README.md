# UFO Raider

> 2157. You're an agent of the Galactic Harvest Bureau. The more you take, the harder they come for you.

**UFO Raider** is a synthwave arcade side-scroller about human abduction — flying a Class-9 Extractor through five hostile sectors while Earth's military response escalates in real time.

## What It Is

A browser-based arcade game set in 2157. Players pilot a UFO through parallax-scrolling environments across five sectors, abducting humans while managing a GTA-style wanted level that ramps up military response the higher your count climbs. Combo multipliers up to ×8 reward skill and aggression. The longer you stay, the worse it gets — and the better it feels.

## Features

- **Five distinct sectors** with parallax scrolling environments
- **GTA-style wanted level system** — escalating military response tied to abduction count
- **Combo multiplier system** — up to ×8 for skilled play
- **Synthwave visual aesthetic** — distinct palette across sectors
- **Leaderboard system** — compete for top extraction counts
- Single-file HTML — no install, no account, no data collection

## Genre & Tone

Arcade / Side-Scroller / Synthwave  
Satirical / Escalating / High-Score Driven
